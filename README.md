# Tucil1_QueensLinkedIn

### Neswa Eka Anggara | 13524136

Sebuah program untuk mencari penyelesaian dari permainan Queens pada LinkedIn menggunakan algoritma Brute Force.

## Requirements

### Java
- **Version:** 17 or higher
- **Download links:**
  - [Oracle JDK 17](https://www.oracle.com/java/technologies/downloads)

### Maven
- **Version:** 3.2.5 or higher (recommended 3.6.3+)
- **Download links:**
  - [Direct Apache Maven Official Downloads](https://dlcdn.apache.org/maven/maven-3/3.9.11/binaries/apache-maven-3.9.11-bin.zip)

## How to run
1. Compile
```bash
mvn clean package
```
or
```bash
mvn clean compile
```

2. Run
```bash
mvn exec:java -Dexec.mainClass="stima.App"
